# Trigger Graphs Datalog Engine

Python bindings for [GLog](https://github.com/karmaresearch/glog), a [Datalog](https://en.wikipedia.org/wiki/Datalog) engine based on trigger graphs. Originally based on the [glog-python](https://github.com/karmaresearch/glog-python) bindings.

### Installation
 
```bash
pip install git+https://github.com/jjcmoon/triggergraphs
```


### Papers

The inner workings of the GLog engine are described in the following papers.

- [Materializing knowledge bases via trigger graphs](https://dl.acm.org/doi/abs/10.14778/3447689.3447699)
- [Probabilistic Reasoning at Scale: Trigger Graphs to the Rescue](https://dl.acm.org/doi/abs/10.1145/3588719)
